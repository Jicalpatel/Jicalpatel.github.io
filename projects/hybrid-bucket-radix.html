Spring 2025 Algorithms Performance

Hybrid Bucket-Radix Sorting

A performance study comparing non-comparison sorting algorithms on GPU and CPU architectures, with attention to scalability and memory behavior.

Project Story

This project studies how sorting performance changes across architectures and algorithm families. The hybrid bucket-radix approach combines distribution-based partitioning with digit-wise sorting to improve performance under selected data distributions.

The strongest portfolio angle is not only the algorithm name, but the engineering mindset: measure carefully, compare fairly, and explain trade-offs between parallel execution, memory movement, and data shape.

What makes it strong

  • Highlights algorithmic thinking with practical benchmarking.
  • Connects CPU and GPU architecture behavior to real performance outcomes.
  • Shows comfort with technical writing, complexity analysis, and experiment design.
