Skin Disease Identification

An AI-assisted skin disease identification project built with TensorFlow, Keras, and OpenVINO, paired with a mobile-friendly diagnosis UI.

Project Story

This project focuses on making early skin disease identification more accessible through a machine learning model and a mobile-friendly user interface.

The resume highlights a 75% accuracy AI model built with TensorFlow, Keras, and OpenVINO, then deployed through a UI designed for easier diagnosis access.

What makes it strong

  • Built an AI model with 75% accuracy.
  • Used TensorFlow, Keras, and OpenVINO for model development and optimization.
  • Deployed a mobile-friendly interface for accessible diagnosis support.
