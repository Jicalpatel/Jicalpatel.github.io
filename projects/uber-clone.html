A blockchain-based ride-hailing system using smart contracts, crypto payments, real-time GPS tracking, and automated fare calculation.

Project Story

This project translates the familiar ride-hailing workflow into a blockchain-backed application concept. It uses smart contracts and Ethereum Rinkeby Testnet payment flow to explore trust, transaction handling, and ride payment automation.

The project also includes practical ride-hailing features such as real-time GPS tracking and automated fare calculations, connecting product UX with technical system design.

What makes it strong

  • Developed a blockchain-based ride-hailing system using smart contracts.
  • Integrated crypto payments using Ethereum Rinkeby Testnet.
  • Included real-time GPS tracking and automated fare calculation logic.
