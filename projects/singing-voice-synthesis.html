Singing Voice Synthesis

An end-to-end machine learning system that converts lyrics into sung audio using a two-stage acoustic model and neural vocoder pipeline.

Project Story

This project explores how modern speech and audio models can generate more natural singing voices. Instead of manually coding vocal rules, the system uses transfer learning to move from text and pitch toward mel-spectrograms, then converts those spectrograms into playable waveforms.

The pipeline uses FastSpeech2 Conformer for acoustic modeling and HiFi-GAN as the vocoder. The work also includes exploratory data analysis on the Children's Song Dataset, covering pitch distribution, note duration, syllable frequency, mel-spectrograms, and F0 contours.

What makes it strong

  • Built a working text-to-audio inference path with HuggingFace models and PyTorch.
  • Created a Gradio interface so non-technical users can type lyrics and hear generated singing.
  • Connected model output with evaluation artifacts such as spectrogram and waveform comparisons.
